# ⏰ Alarm Clock

A modern and interactive Alarm Clock built using **HTML**, **CSS**, and **JavaScript**. This project allows users to set multiple alarms, view the current time in real-time, and manage alarms with an intuitive interface.

## 🚀 Live Demo

🌐 **https://js-advance-projects-pomy.vercel.app/**

## ✨ Features

- ⏰ Real-time digital clock
- ➕ Set multiple alarms
- 🔔 Alarm notification with sound
- ❌ Delete alarms individually
- 🎨 Clean and responsive UI
- 📱 Works on desktop and mobile browsers

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)

## 💡 How It Works

1. Select the hour, minute, second, and AM/PM.
2. Click **Set Alarm**.
3. The alarm is added to the list.
4. When the selected time matches the current time, the alarm rings.
5. Delete an alarm anytime using the delete button.
