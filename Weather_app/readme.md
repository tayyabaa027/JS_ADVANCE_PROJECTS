# 🌦️ Weather App

A modern and responsive Weather App built using **HTML**, **CSS**, and **JavaScript**. It fetches real-time weather information using the **OpenWeatherMap API** and displays the current weather conditions of any city entered by the user.

## ✨ Features

- 🔍 Search weather by city name
- 🌡️ Real-time temperature in Celsius

## ⚙️ How It Works

1. Enter a city name.
2. Click the search button.
3. The app sends a request to the OpenWeatherMap API.
4. Weather data is fetched and displayed instantly.
5. The weather icon changes automatically according to the current weather condition.

---

## 🔑 API Used

**OpenWeatherMap API**

The application uses the Current Weather Data API to fetch live weather information.
